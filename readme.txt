== Description ==

Adding tracking code to all sites with the ability to select the placement of tracking code in the header or footer.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `dc-google-analytics` to the `/wp-content/plugins/` directory
1. Network activate the plugin through the 'Plugins' menu in WordPress Network Admin
1. Add your tracking code and save some settings, 'Network Admin > Settings'.
